#### Install

> **NOTE** At least Python3.6 is needed

Install libraries dependencies

```
pip3.6 install --user -r  Flexible_Network/requirements.txt
```



#### Run unit tests

```
export PYTHONPATH=Flexible_Network/
python3.6 -m pytest tests/  -v
```
